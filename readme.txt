== Description ==

**Webp Converter Pro** is the ultimate tool for optimizing your WordPress media library. It automatically converts your existing JPEG and PNG images into the modern, high-performance WebP format.

== Installation ==

1. Upload the `webp-converter-pro` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the **Webp Converter** menu in your dashboard.
4. Select images and click "Convert".

== Frequently Asked Questions ==

= Is my data safe? =
Absolutely. The conversion process happens entirely on your hosting server. We do not have access to your images, and they are never uploaded to any external cloud service.

= Does this require a subscription? =
No. Since the plugin uses your server's local PHP GD or Imagick library, you don't need to pay for external API credits.

= Does this delete my original images? =
Yes, to save server space, this plugin replaces the original file with the WebP version.
